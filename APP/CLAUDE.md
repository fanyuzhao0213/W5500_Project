# Claude Agent Rules

你是一个专业嵌入式开发Agent。

当前项目：

* STM32F405RGT6
* W5500
* MQTT
* FreeRTOS
* Keil MDK5

---

# 必须遵守

## 1. 必须使用HAL库

禁止：

* 标准库
* LL库
* 裸寄存器（除特殊要求）

---

## 2. 所有代码必须完整输出

禁止：

```c
// 用户自行实现
```

必须给完整代码。

---

## 3. 所有函数必须有错误处理

禁止：

忽略返回值。

必须：

```c
if(ret != 0)
{
    LOGE(...);
}
```

---

## 4. 禁止动态内存

禁止：

* malloc
* calloc
* realloc
* free

原因：

工业项目长期运行禁止堆碎片。

---

## 5. FreeRTOS要求

禁止：

* 长时间阻塞
* 死循环等待
* while(socket wait)

必须：

所有操作支持超时。

---

## 6. 网络要求

禁止：

MQTT失败时重置整个W5500。

必须：

分层恢复：

* Link层
* TCP层
* MQTT层

---

## 7. 日志规范

统一使用：

```c
LOGI(...)
LOGW(...)
LOGE(...)
```

禁止：

printf。

---

## 8. 编码格式

必须：

UTF-8。

禁止：

乱码中文。

---

## 9. 代码风格

函数名：

```c
snake_case
```

变量：

```c
g_global_var
s_static_var
```

宏：

```c
UPPER_CASE
```

---

## 10. 任务设计

禁止：

一个任务负责所有网络逻辑。

必须：

分层任务。

---

# 代码生成要求

生成代码时：

* 优先稳定性
* 优先工业级设计
* 优先长期运行
* 优先异常恢复

禁止：

Demo风格代码。
